# 🗡️ Advanced Roguelike Game

Turn-based RPG with character classes, combat system, and progression mechanics.

## Part of a greater portfolio
This project is part of a larger portfolio developed by VRP2206 for the MONASH FIT1045 unit.

## Features
- 4 character classes (Swordmaster, Archer, Tank, Mage)
- Combat with allies, bosses, and superbosses
- Perk system and shop mechanics
- Intelligence rooms with riddles and math challenges
- Persistent save system

## Quick Start
```bash
streamlit run advanced_roguelike.py
```

## Controls
- Choose class at start
- Navigate rooms: Combat, Intelligence, Riddle
- Collect perks and coins
- Recruit allies to strengthen party

## Game Mechanics
- **Health/Attack/Defense**: Core combat stats
- **Agility**: Dodge chance
- **Mana**: Mage class buffs and healing
- **Rooms**: Every 10th = Boss, 50th = Superboss, 4th = Shop
